## see all commits

    $ git log --all

## See last 10 commits

    $ git log -n 10

## see branches

    $ git log --decorate

## one line

    $ git log --oneline

## filter by author

    $ git log -n 10 --author=<author name>

## graph representation

    $ git log --graph

## no pager

    $ git --no-pager log -n 10
