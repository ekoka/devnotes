### standard `for` loop

    for (initialize counter ; check condition ; mutate counter) {
        ...
    }

    // e.g.
    
    for (var i = 0; i < 10; i++ ) {
        ...
    }

### `for-in` loop

    for (item in collection) {
        ...
    }

### `while` loop

    while(condition()){
        
    }

### `do-while` loop

    do {
    } while (condition())
