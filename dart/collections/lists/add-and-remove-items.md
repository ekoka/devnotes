## Adding items

### create with items

    var fruits = ['apple', 'oranges'];

### add an item

    fruits.add('kiwis')

### add multiple items

    fruits.addAll( ['grapes', 'banana'] )


## Removing items

### remove an item at specific index

    // remove third item
    final kiwi = fruits.removeAt(2);

### empty the list

    fruits.clear();
